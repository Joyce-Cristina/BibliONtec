    Administrador
    O administrador é o usuário com o mais alto nível de permissão dentro do sistema BibliONtec.
    Ele é responsável pela gestão geral da plataforma, controle de instituições, usuários, funcionários, relatórios, backups e atualizações. Enquanto o bibliotecário gerencia o acervo e os empréstimos, o administrador supervisiona a estrutura técnica e institucional do sistema.

    Bibliotecário
    🔹 Etapas do processo:

    1° Ao clicar em “Usuário”, o sistema exibe um formulário padrão com os seguintes campos iniciais:

    🔹 Nome completo, E-mail institucional, Telefone, Senha e Foto.

    2° O bibliotecário deve definir o tipo de usuário por meio de uma caixa de seleção (checkbox):

    🔹 Se marcar “Aluno” → são exibidos os campos adicionais:

    Série e Curso.

    🔹 Se marcar “Professor” → os campos “Série” e “Curso” desaparecem, pois não se aplicam a esse tipo de usuário.

    3° Após o preenchimento, o bibliotecário confirma o cadastro.

    4° O sistema gera automaticamente um login com o e-mail institucional e uma senha temporária, que o usuário poderá alterar no primeiro acesso.
    Acesso:

    - Etapas do processo:

    1° Ao acessar a aba “Cadastro”, o bibliotecário seleciona “Livro”.

    2° O sistema exibe um formulário com todos os campos necessários para o registro:
    🔹 Título, Autor(es), Código do autor, Edição, Editora, Local de publicação, Data de publicação, Número de páginas, Assunto, Linguagem da catalogação, Classificação (CDD), Área de conhecimento, ISBN, Sinopse, Capa e Disponibilidade.

    3° O sistema possui uma função automatizada de preenchimento via ISBN:
    🔹 Ao digitar o número do ISBN e clicar em “Buscar informações”, o sistema faz a busca automática dos dados do livro (como título, autor, editora e capa).
    🔹 Caso algumas informações não sejam encontradas, os campos permanecem em branco para que o bibliotecário complete manualmente.

    4° Após o cadastro, o sistema gera automaticamente uma etiqueta contendo:
    🔹 Código CDD, Código do autor, Área de conhecimento e Código de barras.

    5° O bibliotecário pode baixar ou imprimir a etiqueta imediatamente após o registro.
